# todolist
A CLI/GUI todo list developed with Maven, Jenkins and a lot of friends

## Setup

  - Clone the repo
  - Create a `.env` file with API tokens by Coveralls and Sonarcloud
  - `docker-compose up`
  - Visit `localhost:8090` for the Jenkins web interface and launch jobs.
